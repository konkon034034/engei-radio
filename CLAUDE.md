# engei-radio プロジェクト仕様書

## AI作業ルール

- **Thinking（思考プロセス）は必ず日本語で書く。英語は絶対禁止**
- 未使用コードの残骸は即座に削除する

### 会話引き継ぎルール

- **task.mdに作業状態を常に記録する**
- **walkthrough.mdに完了作業の成果を記録する**

## 概要

もう枯らさない!家庭園芸チャンネル。紙芝居形式の擦り直し動画を自動生成。

## 動画形式

こすり直し紙芝居（人気YouTube動画の視点変更リメイク）

### 3幕構成（全紙芝居チャンネル共通・2026-02-19〜）

| 幕 | 内容 |
|:---|:---|
| **OP（hook）** | **衝撃の事実フック**: クイズではなく事実。大きな数字をグラフ/チャートで表示して視聴者を釘付け |
| **本編（causes〜summary）** | チャート・折れ線グラフ・棒グラフ等を多用。人気動画のデータを視覚化。カツミとヒロシが掛け合いで解説 |
| **控え室（hikaeshitsu）** | **エピローグ余韻**: 本編の衝撃を振り返り、余韻に浸れる温かいトーク |

## 絶対ルール

1. **動画生成はRemotion専用**
2. **動画は3分以上**
3. **TTS優先順位**: Edge TTS → Amazon Polly
4. **カツミ・ヒロシの人間性**: 体験談・共感・庶民あるある
5. **OPスライドは衝撃の事実で釘付けにする**（クイズ不可）
6. **控え室はエピローグ余韻**（本編の振り返り+温かい締め）
7. **LLMモデル**: gemini-2.5-pro（ウルトラプラン中）
